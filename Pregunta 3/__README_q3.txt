****************************
Test SD - Pregunta 3
****************************

Lenguaje esperado: JS, CSS, HTML
Complejidad: Fácil 

--- Descripción ---

Crea una página web sencilla que muestra información sobre diferentes países de América Latina en forma de tabla de datos. 
La tabla debe tener al menos cinco columnas: nombre del país, capital, población, área y idioma oficial.

Los datos de la tabla deben ser proporcionados por el archivo data.json que se cargará mediante una solicitud AJAX. 
La tabla debe tener una barra de búsqueda que permita filtrar los datos por el nombre del país.

Para completar este ejercicio, deberás utilizar HTML y CSS para crear la estructura y el diseño de la página, y JavaScript para realizar la solicitud AJAX, procesar 
los datos del archivo JSON y crear y manipular la tabla dinámicamente.
