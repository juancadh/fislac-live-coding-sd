Lenguaje esperado: Python
Complejidad: Intermedio 

--- Descripción ---

Realiza una función que tome un arreglo de números enteros y regrese un arreglo de dos elementos, 
los cuales representan el rango de números más largo que contiene el arreglo. 
El primer elemento del input debe ser el límite inferior y el segundo elemento debe ser el límite mayor del rango.


--- Ejemplo ---

Si el arreglo de números es [4,7,3,2,8,1] entonces el resultado debe ser [1, 4]. Es decir, un rango de tamaño 4.


--- Notas ---

* El arreglo de entrada es una lista no ordenada de enteros.
* Asumir que solo existe un rango mayor.
* Los numeros no necesitan ser ordenados para lograr encontrar el resultado.
* Puedes utilizar cualquier tipo de estructura de datos que desees. (Recomendado: Hash Table)
